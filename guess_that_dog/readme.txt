Hello! This executable is for this little game I've made called Name That Dog! It's based on the Saturday Night Live sketch of the same name.

There's not much to it. You are shown a dog and you have to guess what the dog's name is.

If you want to play it, by all means you may download and play it. If you want to take a look at the source code, peek inside the src folder and the code is there in main.rs.

This is still a work in progress. There are a few things I want to change, like making names not case sensitive. But in the meantime, it's a silly little thing that I did for Rust practice.
